# BossyWvsP
Skrypty na bossy z Wojownicy vs Potwory III

Aby pobrać skrypty, kliknij zielony przycisk "Code" i wybierz "Download ZIP".

Aby skrypty działały poprawnie:

	W server.properties zmień allow-flight=false na true
	W plugins/WorldGuard/config.yml zmień false na true w:
	   	block-wither-skull-explosions; 
	  	block-wither-skull-block-damage; 
	  	block-other-explosions; 
	  	block-fireball-explosions; 
	  	block-fireball-block-damage; 
	W plugins/Stacksize/config zmień "SPLASH_POTION:" z 4 na 64
